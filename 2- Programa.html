El salto de linea se puede colocar dentro de los scripts. O sea que se
puede usar codigo HTML dentro del script de JS, trabajando de forma dinamica.

Ejemplo de salto de linea dentro del script.

Se coloca el TAG -br> dentro de la orden....documen.write("La edad de mateo es...-br>")
O tambien se le puede agregar una linea...document.write("-br>")


Si en la linea document.write("?") quito el ? y coloco un numero "x", me mostraria la edad.
Si le saco las comillas al numero "x", tambien funciona perfectamente.
Al estar el numero entre comillas, JS lo toma como string, y sin comillas es un numero.
Tambien se puede hacer calculos matematicos dentro de las comillas, por ejemplo:
se puede hacer (10+15) y dara el resultado correspondiente (25).
Tambien se puede poner un string mas un numero, por ejemplo:
Se puede colocar("10" + 15), y me da como salida 1015, toma el string "15" y le agrega el num 10
